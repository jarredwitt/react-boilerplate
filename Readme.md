# Webpack + Babel + React + React-Router + Redux

## Contains

- [x] Babel
- [x] CSS Modules
- [x] PostCSS
- [x] React
- [x] React Router
- [x] Redux
- [x] Webpack 2

Make sure you have the latest node before using Webpack 2. Some depenencies my fail in older versions. 

## Setup

```
$ npm install
```

## Running

```
$ npm start
```

## Build

```
$ npm run build
```
